# 📊 E-Commerce Revenue Insights Dashboard

An interactive Power BI dashboard designed to track sales performance, identify growth trends, and uncover key revenue drivers across categories, products, and geographies.

---

## 🚀 Project Overview

This project transforms raw eCommerce transactional data into actionable business intelligence using Power BI, DAX, and Power Query.

It provides insights into:
- Sales performance tracking
- Revenue trends over time
- Product and category analysis
- Geographic sales distribution
- Growth metrics (YoY and MoM)

---

## ✨ Key Features

### 📈 Dynamic Time Intelligence
Switch between Day, Month, Quarter, and Year views using advanced DAX measures for flexible reporting.

### 🤖 AI-Generated Insights
A smart narrative box automatically summarizes:
- YoY performance changes
- MoM growth trends
- Top-performing products

### 📊 Growth Tracking
- Year-over-Year Growth: **93.2%**
- Month-over-Month Growth: **3.57%**

### 🌍 Geospatial Analysis
Interactive map visualizations showing revenue distribution by city and region.

### 📦 Product & Category Analysis
Identify top-performing and underperforming products across categories.

---

## 🛠️ Tech Stack

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query (M Language)
- Star Schema Data Modeling
- Claude AI (MCP Server for DAX optimization)

---

## 🧠 Data Model

The dashboard follows a **Star Schema** design.

### Fact Table:
- Sales Transactions (Revenue, Quantity, Order ID)

### Dimension Tables:
- Date Table (Time Intelligence)
- Product Table (Category, Subcategory)
- Customer Table (Segment, Region)
- Geography Table (City, State, Country)

---

## 📌 Key DAX Measures

- Total Sales
- Revenue MTD / QTD / YTD
- YoY Growth %
- MoM Growth %
- Dynamic Average Sales
